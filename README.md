# Desafio de Código - Contador

Este projeto é uma resolução do desafio proposto na trilha backend com Java do Bootcamp Santander 2024. Este é um programa em Java que recebe dois números inteiros do usuário e conta de um número até o outro.

## Como usar

1. Clone este repositório para sua máquina local.

```bash
git clone https://github.com/erickallanss/dio-santander-java-contador

cd DesafioControleFluxo

javac -d bin src/*.java

java -cp bin Contador

```

## Funcionamento

O programa solicita dois números inteiros do usuário, sendo o primeiro menor que o segundo. Em seguida, ele conta de forma crescente do primeiro número até o segundo, imprimindo cada número no console.
